# Installation

Just install from PyPI:

```pip install sample-factory```

SF is known to work on Linux and macOS. There is no Windows support at this time.

### Environment Support

To run Sample Factory with one of the available environment integrations, please refer to the corresponding documentation sections: 

- [Mujoco](../environment-integrations/mujoco.md)
- [Atari](../environment-integrations/atari.md)
- [ViZDoom](../environment-integrations/vizdoom.md)
- [DeepMind Lab](../environment-integrations/dmlab.md)
- [Megaverse](../environment-integrations/megaverse.md)
- [Envpool](../environment-integrations/envpool.md)
- [Isaac Gym](../environment-integrations/isaacgym.md)

Sample Factory allows users to easily add custom environments and models, refer to [Customizing Sample Factory](customizing.md) for more information.
